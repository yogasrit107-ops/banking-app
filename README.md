# 🏦 Mini Banking App

A simple full-stack banking application built using:

* 🔹 Backend: ASP.NET Core Web API
* 🔹 Frontend: HTML, JavaScript
* 🔹 Features: Login, Balance Check, Deposit, Withdraw

---

## 🚀 Features

* ✅ User Login
* 💰 Check Balance
* ➕ Deposit Money
* ➖ Withdraw Money
* 🔐 Basic Authentication

---

## 🛠️ Tech Stack

* ASP.NET Core
* C#
* HTML
* JavaScript

---

## ▶️ How to Run

### 1️⃣ Backend

```bash
cd backend
dotnet run
```

Runs on: http://localhost:5127

---

### 2️⃣ Frontend

Open `frontend/index.html` using Live Server
